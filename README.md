# PhotoBrowser Enigma2 v1.0

Advanced Photo Browser for Enigma2 receivers.

Created by theking-cs

---

## Features

✔ Browse HDD / USB / Internal folders  
✔ Fullscreen image viewer  
✔ Zoom support  
✔ Image rotation  
✔ Preview thumbnails  
✔ Remote control navigation  
✔ Modern FullHD interface  
✔ Supports JPG / JPEG / PNG  

---

## Supported Images

- OpenATV
- OpenPLI
- Egami
- OpenVision
- Python2 / Python3 images

---

# Installation

Connect to your receiver using SSH/Telnet and run:

bash wget -qO- https://raw.githubusercontent.com/theking-cs/PhotoBrowser/main/installer.sh | sh 

---

# Manual Installation

Copy plugin folder to:

bash /usr/lib/enigma2/python/Plugins/Extensions/ 

Restart Enigma2 after installation.

---

# Controls

| Key | Action |
|------|--------|
| OK | Open image / Fullscreen |
| EXIT | Back / Exit |
| RED | Rotate image |
| YELLOW | Zoom in |
| BLUE | Zoom out |
| LEFT/RIGHT | Previous / Next image |
| UP/DOWN | Move image while zoomed |
